Windows Computer

  1. Navigate to the remote access portal (Remote Access Portal)
  2. In the buttom right of the page you should see Citrix download
  3. Click "Windows Citrix Download"
  4. Click "Citrix Workspace app 2409.10 for Windows"
  5. Press the top Download File button
  6. Once Downloaded click on the file to run Citrix installer
  7. Follow the prompts for the installer
  8. Once install it should ask for a store URL ( You can close the application )
  9. Navigate back to the remote access portal
  10. Click Citrix Application top left
  11. Type in username and password
  12. Accept the DUO push
  13. Click Detect Citrix Workspace
  14. Allways allow the website to open Citrix Workspace
  15. Click on Outlook OWA
  16. Login into Email
