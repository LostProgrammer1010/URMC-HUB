Adding Shared Mailbox Outlook Web

  1. Right click on folders
  2. Click "Add shared folder or mailbox"
  3. Search for the name of the shared mailbox
  4. Press add
  5. Shows up on left as another mailbox

Adding Share Email Outlook Desktop

  1. Open Outlook Application
  2. Click File top left
  3. Account Settings> Account Settings
  4. Click Data Files Tab
  5. Double click email
  6. Advance Tab
  7. Add
  8. Type in the shared email full name “name@URMC.rochester.edu”
  9. Press Ok> Apply
