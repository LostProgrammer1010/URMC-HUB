Providing Remote Access

  1. Open Active Directory, and search for the PC. Right Click that PC, and select Manage.
  2. Navigate to Local Users and Groups, select Groups, then select Remote Desktop Users.
  3. Here you can see which users have the permission to RDP into the PC. If none are listed, you can add them by selecting Add.
    4. ⚠️Only one person per computer, if there is already a person list send to CTS to have them decide
